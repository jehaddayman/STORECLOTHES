## 🐛 حل المشاكل الشائعة

### المشكلة: "لا توجد بيانات"

**الحل:**
1. اضغط "ملء البيانات"
2. انتظر الرسالة
3. أعد تحديث الصفحة (F5)

### المشكلة: "لا يظهر الموقع"

**الحل:**
```bash
# تأكد أن docker-compose يعمل
docker ps

# إذا مش شغال
docker-compose up

# انتظر 30 ثانية
# افتح: http://localhost:3000
```

### المشكلة: "Database error"

**الحل:**
```bash
# أوقف
docker-compose down

# امسح البيانات
docker volume rm clothing-erp_mongodb_data

# شغّل مرة أخرى
docker-compose up
```

### المشكلة: "Port مشغول"

**الحل:**
```bash
# اقتل العملية القديمة
docker-compose down

# شغّل مرة أخرى
docker-compose up
```
