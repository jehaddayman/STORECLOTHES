## 🚀 تشتغل الآن؟

### اختيار 1: Docker (الأسهل - 5 دقائق)
```bash
docker-compose up
# بس! الموقع بيفتح تلقائياً
```

### اختيار 2: يدوي (15 دقيقة)
```bash
# Backend
cd backend
npm install
npm start

# Frontend (ترمينل جديد)
cd frontend
npm install
npm run dev
```
